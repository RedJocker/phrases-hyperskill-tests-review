Description 

To see the phrases that we have added to the application, we need to print them somewhere on the screen. In this stage, you will set up the RecyclerView so that it shows the list of the phrases on your Activity screen.

Objectives

  1. Create a XML layout for your RecyclerView items. The view should contain a TextView with a phrase (ID: phraseTextView) and a "Delete" TextView to remove it (ID: deleteTextView). Your design still can be whatever you want, but keep these two Views with the ID's mentioned earlier.
  2. Create an adapter for your RecyclerView. Adapter should show the list with your predefined phrases. Set an OnClickListener for the "Delete" TextView which removes the item. It's not important in this step to save the user's actions in any way, so it's okay if after app restart all the deleted phrases are coming back.
  3. You should show at least 3 phrases on the screen in this stage.
