Description

In case you would like to have some motivation boost in mornings, you can create an app which sends you some random phrases! In this stage, you will design an UI for this application. There will be a list of phrases, a button to add more, and a text with reminder time.

Objectives

Create a screen which contains the following components:

  1. A TextView that shows whether the reminder is set, and if it is, at what time
  2. A RecyclerView that will hold your phrases
  3. A Floating Action Button which allows you to add more phrases

Set these ID's for your widgets: TextView: reminderTextView, RecyclerView: recyclerView, Floating Action Button: addButton.

reminderTextView text attribute should be set to "No reminder set".
